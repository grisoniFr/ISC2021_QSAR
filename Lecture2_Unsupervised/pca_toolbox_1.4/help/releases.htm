What's new in version 1.4

The graphical interface has been improved. Now forms with plots can be resized. Classes can be loaded both as numerical or string arrays. Visualisation of class potential in the score plots was added.

[-> top]

What's new in version 1.3

Normalised T2 Hotelling and Q residuals were added [Journal Of Chemometrics, 2000; 14: 725–736]. These can be accessed in the Principal Component Analysis results.

[-> top]

What's new in version 1.2

Minor bugs were fixed. Forms were modified in order to get the same dimension on different Matlab versions.

[-> top]


What's new in version 1.1

Minor bugs were fixed. Some forms (variable profiles and optimal component selection) were adapted to work on Matlab 2015.

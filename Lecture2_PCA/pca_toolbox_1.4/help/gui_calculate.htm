Principal Component Analysis (PCA)

In order to calculate models based on Principal Component Analysis (PCA), select "calculate->PCA". The form for PCA options will appear. Here you can select the data scaling and the number of optimal components to be retained in the PCA model.
The optimal number of Principal Components can be estimated by choosing "calculate->optimal number of components for PCA". This will open a form to set options (data scaling, number and type of cross validation groups) for the calculation. In particular, cross validation can be performed with venetian blinds or contiguous blocks. Regarding venetian blinds, with 3 cross-validation groups the split of the first group in venetian blinds will be [t,0,0,t,0,0,....,t,0,0], while the second one will be [0,t,0,0,t,0,....,0,t,0], and so on. On the other hand, the split of the first group with contiguous blocks will be [t,t,t,t,0,0,....,0,0,0] and so on. The number of cross-validation groups can be defined (note that a number of groups equal to the number of samples corresponds to the leave-one-out validation procedure).
A new form with the results of PC selection will appear. Here it is possible to plot different parameters (eigenvalues, explained variance, cumulative explained variance, rmsecv, IE and IND) as a function of the PCs included in the model. The number of PCs to be shown in the plot can be changed. All plots can be exported as MATLAB figures and values of eigenvalues can be seen by means of the "view eigenvalue" button. When plotting eigenvalues, two horizontal lines are drawn; these correspond to the average eigenvalue and to the average eigenvalue multiplied by 0.7; moreover, the number of components suggested by eigenvalue-based methods (AEC, CAEC, KL, KP) are labelled. See the example provided in this help to better understand.

Cluster Analysis

Hierarchical Cluster Analysis can be calculated by selecting "calculate->hierarchical cluster analysis". The form for settings will appear. Here you can select the data scaling, the type of distance and type of linkage.

Multidimensional Scaling (MDS)

Select "calculate->MDS" to calculate Multidimensional Scaling (MDS). The form for settings will appear. Here you can select the data scaling and the type of distance to be used.

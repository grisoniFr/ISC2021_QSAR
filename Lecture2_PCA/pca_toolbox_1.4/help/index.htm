What PCA toolbox for MATLAB is?

The PCA toolbox for MATLAB is a collection of MATLAB modules for calculating unsupervised multivariate models for data structure analysis: Principal Component Analysis (PCA), Multidimensional Scaling (MDS) and Cluster Analysis.

This is the version 1.4 of the PCA toolbox for MATLAB

PCA toolbox for MATLAB has been released by Milano Chemometrics and QSAR research Group

Installing the toolbox

MATLAB should be installed, while the Statistics Toolbox is needed to compute Cluster Analysis and Multidimensional Scaling. In order to install the toolbox, simply copy the files to a folder (e.g. "PCA toolbox for MATLAB"). Then, in order to use it, select the same folder as MATLAB current directory. The toolbox was built on MATLAB 2016. MATLAB routines were tested on previous versions until MATLAB 2010, the graphical interface was tested on MATLAB versions until MATLAB 2014.

Principal Component Analysis results

Eigenvalues (as well as explained and cumulative variances) can be analysed or plotted by selecting "results->PCA results->view/plot eigenvalues". PCA scores and loadings can be analysed by selecting "results->PCA->results->scores" and "results->PCA->results->loadings", respectively.
Results related to samples (such as scores, Q residuals and T2 Hotelling) are visualised in a form, where the user can select what to plot by means of specific menus ("x axis" and "y axis"). In this plot, samples are coloured on the basis of their experimental class or response (if loaded). Plots can be exported as MATLAB figures ("export plot" button). Legend, sample labels and convex hulls of classes (if loaded) can be added to the plots with the corresponding check boxes.
By selecting a reference class from the menu "class potential", one can highlight the class potential, which is defined on the basis of a gaussian kernel calculated with the distribution of the class samples in the visualised plot, as shown in the following figure for the first two PC for the blue class.


The "view sample" button enables the user to select a specific point in the score plot: profiles of the values of the raw or scaled variables of the selected sample, Q residuals contributions and Hotelling T2 contributions (raw or normalised) will appear in a new window, as shown in the following figure. Normalised Q residuals contributions and Hotelling T2 contributions are calculated on the basis of the following scientific paper: A.K. Conlin, E.B. Martin, A.J. Morris, Confidence limits for contribution plots, Journal Of Chemometrics, 2000; 14: 725–736.


Loadings are visualised in a form, where the user can select what to plot by means of specific menus ("x axis" and "y axis"). Plots can be exported as MATLAB figures ("export plot" button). Variable labels can be added to the plots with the corresponding check box.

Cluster Analysis results

The dendrogram calculated by means of Cluster Analysis can be analysed by selecting "results->view dendrogram". A new form will appear. Here it is possible to analyse the dendrogram, set the number of clusters and save the cluster assignations as a vector.

Multidimensional Scaling results

MDS coordinates of samples can be analysed by selecting "results->MDS results->scores". A form (as the one where PCA scores are visualised) will appear. In this plot, samples are coloured on the basis of their experimental class or response (if loaded). The user can modify the coordinates to be analysed on the plots. Plots can be exported as MATLAB figures. Finally, the "view sample" button will enable to select a single point on the plot and subsequently view the raw and scaled variable profiles of the selected sample.

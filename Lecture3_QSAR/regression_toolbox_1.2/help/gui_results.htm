Regression performances

It is possible to analyse the regression performances of the calculated model by clicking "results->regression measures ". The following form will appear:


R2 and RMSE are shown both in fitting (all the samples used to build the model) and validation (look here for further information on these measures).
The buttons "view measures" and "view predictions" open new variables in the MATLAB workspace with the corresponding values: predicted response in fitting and cross validation and regression measures in fitting and cross validation.

Results related to samples

Results related to samples (such as experimental vs predicted response, residuals, leverages, scores, Q residuals, Hotelling T2, etc...) are visualised in a form, where the user can select what to plot by means of specific menus ("x axis" and "y axis"). Samples can be coloured on the basis of their experimental response on a grey scale. Plots can be exported as MATLAB figures ("export plot" button). Sample labels can be added to the plots with the corresponding check box.


The "view sample" button enables the user to select a specific point in the sample plot: profiles of the values of the raw or scaled variables of the selected sample, Q residuals contributions and Hotelling T2 contributions (raw or normalised) will appear in a new window, as shown in the following figure. Normalised Q residuals contributions and Hotelling T2 contributions are calculated on the basis of the following paper: A.K. Conlin, E.B. Martin, A.J. Morris, Confidence limits for contribution plots, Journal Of Chemometrics, 2000; 14: 725–736.

Results related to variables

Results related to variables (such as coefficients, standardised coefficients and loadings) are visualised in a form, where the user can select what to plot by means of specific menus ("x axis" and "y axis"). Plots can be exported as MATLAB figures ("export plot" button). Variable labels can be added to the plots with the corresponding check box.

Ordinary Least Squares (OLS) results

Predictions and coefficients of variables associated to OLS can be analysed by selecting "results->Ordinary Least Squares->predictions" and "results->Ordinary Least Squares->coefficients", respectively. These will open new forms where details of the calculated model can be analysed.
In particular, the sample plot will enable the user to analyse predictions, leverages and residuals.
The variable plot will enable the user to analyse coefficients and standardised coefficients.

[-> top]


Partial Least Squares (PLS) results

Scores and loadings of latent variables associated to PLS can be analysed by selecting "results->Partial Least Squares->scores" and "results->Partial Least Squares->loadings", respectively. These will open new forms where details of the calculated PLS model can be analysed.
In particular, the sample (score) plot will enable the user to analyse predictions, residuals, scores on the latent variables, Q residuals, T2 Hotelling, leverages.
The variable (loading) plot will enable the user to analyse coefficients, loadings and weights of variables.
Finally, cumulative and explained variances of the selected latent variables can be visualised by clicking "results->Partial Least Squares->variances".

[-> top]


Principal Component Regression (PCR) results

Scores and loadings of principal components associated to PCR can be analysed by selecting "results->Principal Component Regression->scores" and "results->Principal Component Regression->loadings", respectively. These will open new forms where details of the calculated PCR model can be analysed.
In particular, the sample (score) plot will enable the user to analyse predictions, residuals, scores, Q residuals, T2 Hotelling, leverages.
The variable (loading) plot will enable the user to analyse coefficients and loadings.
Finally, cumulative and explained variances of the selected latent variables can be visualised by clicking "results->Principal Component Regression->variances".

Ridge results

Predictions and coefficients of variables associated to Ridge can be analysed by selecting "results->Ridge regression->predictions" and "results->Ridge regression->coefficients", respectively. These will open new forms where details of the calculated model can be analysed.
In particular, the sample plot will enable the user to analyse predictions, leverages and residuals.
The variable plot will enable the user to analyse coefficients and standardised coefficients.

Local regression results

Predictions associated to local regression, K Nearest Neighbours (KNN) and Binned Nearest Neighbours (BNN), can be analysed by selecting "results->local regression->predictions". This will open the sample plot where the user can analyse predictions and residuals.

[-> top]


Variable selection: Genetic Algorithms (GA) results

GA results can be analysed by selecting "results->variable selection ->plots". This will open a figure with two plots. The first plot collects a bar plot representing the frequency of selection of each variable in the best model of each GA run. The second plot represents, for each GA run, the R2cv and R2 on test samples (if available) acheived by the best model in the GA population.
Finally, "results->Variable selection ->list of selected variables " will open a new window with the list of selected variables.

Variable selection: Forward Selection (FS) and Reshaped Sequential Replacement (RSR) results

Select "results->variable selection ->list of selected variables" to open a new window with the list of variables selected by means of Forward Selection (FS) and Reshaped Sequential Replacement (RSR).

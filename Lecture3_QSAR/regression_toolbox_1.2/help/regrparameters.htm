Regression measures provided in the toolbox

Regression measures are provided as MATLAB structure (reg_param) and can be found in the results section of the GUI, as well as in the results obtained in the command window of MATLAB (see the description of the MATLAB routines). The regression measures can be analysed by typing, in the MATLAB command window:

model.reg_param


when fitting the model, or

cv.reg_param

when validating the model.

This is a list of the main routines you can use to calculate regression models by means of the Regression toolbox for MATLAB:

Open the graphical interface: reg_gui

reg_gui opens the graphical interface (GUI) for calculating all regression models provided in the toolbox; in order to open the graphical interface, just type on the MATLAB command line:

reg_gui

there are no inputs; data can be loaded and saved directly from the graphical interface.

Routines for fitting models

Regression models can be calculated by means of the following routines:
Ordinary Least Squares (OLS): olsfit
Partial Least Squares (PLS): plsfit
Principal Component Regression (PCR): pcrfit
Ridge regression: ridgefit
K Nearest Neighbours (KNN): knnfit
Binned Nearest Neighbours (BNN): bnnfit
The output of the routines collects the calculated response vector [samples x 1], the regression measures in fitting, regression coefficients and other details. Type "help routine_name" on the MATLAB command window for further information.

Routines for validation

Validation of the regression models can be performed by means of the following routines:
Ordinary Least Squares (OLS): olscv
Partial Least Squares (PLS): plscv
Principal Component Regression (PCR): pcrcv
Ridge regression: ridgecv
K Nearest Neighbours (KNN): knncv
Binned Nearest Neighbours (BNN): bnncv
The output of the routines collects the predicted response vector [samples x 1], the regression measures in validation, and other details. Cross validation can be performed with venetian blinds (i.e. with 3 cv groups the split of the first group will be [1,0,0,1,0,0,....,1,0,0] and so on) or contiguous blocks (i.e. the split of the first group will be [1,1,1,1,0,0,....,0,0,0] and so on). Moreover, bootstrap with resampling and validation based on random sampling (montecarlo) of 20% of samples can be calculated. Type "help routine_name" on the MATLAB command window for further information.

Routines for predicting new samples

New samples can be predicted by means of the following routines:
Ordinary Least Squares (OLS): olspred
Partial Least Squares (PLS): plspred
Principal Component Regression (PCR): pcrpred
Ridge regression: ridgepred
K Nearest Neighbours (KNN): knnpred
Binned Nearest Neighbours (BNN): bnnpred
The output of the routines collects the predicted response vector [samples x 1] and other details. Type "help routine_name" on the MATLAB command window for further information.

Routines for optimising models

The cross validation procedure for selecting the optimal parameters (components for PCR, latent variables for PLS, K parameter for Ridge, number of neighbours for K-Nearest Neighbors, alpha value for BNN) can be performed with the following routines:
Partial Least Squares (PLS): plscompsel
Principal Component Regression (PCR): pcrcompsel
Ridge regression: ridgeksel
K Nearest Neighbours (KNN): knnksel
Binned Nearest Neighbours (BNN): bnnsel
The output of the routines collects R2 in cross validation and RMSE in cross validation associated to each parameter value. Type "help routine_name" on the MATLAB command window for further information.

Routines for variable selection

Variable selection based on Forward Selection, Genetic Algorithms and Reshaped Sequential Replacement can be calculated with the following routines
Forward Selection: forward_selection
Genetic Algorithms: ga_selection
Reshaped Sequential Replacement: rsr_selection
The output of these routines collects the selection results. Options for Forward Selection, Genetic Algorithms and Reshaped Sequential Replacement can be defined with the forward_options, ga_options and rsr_options routines, respectively.
Type "help routine_name" on the MATLAB command window for further information.
